# TextRPG
---
스파르타 코딩 클럽에서 제공된 기본 코드을 이용해 만든 c-6조 슉의 팀 프로젝트, TextRPG입니다.


### 프로젝트 소개
---
스파르타 코딩 클럽의 영광스러운 스파르탄 전사가 되어 던전을 탐험하여 부를 쌓아나가세요!
더 많은 부와 명예를 위해 싸우세요!

+ __5.04 ~ 5.06__ : 스크립트 병합 및 발표 준비
